Installation Guide

Getting started is easy. Follow these simple steps:

  1. Download the installer: Latest Release
    Optional: Rename the "Custom" folder in your "btdaoc\ui" to "backup"
  2. Extract the contents from the bahui0.91b.zip file to your "btdaoc\ui" folder.
    Overwrite any files when asked, backups are included.
  3. Install Reshade: pick directx 9, when asked if you want to load an ini file, select the "bah-shade.ini" file in your "btdaoc\ui"
    This will install the shaders needed, and nothing else. Should run smooth on any* system.
